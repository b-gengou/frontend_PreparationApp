# Documentation Frontend – PreparationApp

## Introduction
Ce document décrit l'architecture et les technologies du **frontend** de l'application **PreparationApp**, développée avec **React 18.x**, **TypeScript** et **Vite**.

---

## Technologies Utilisées

| Technologie          | Version | Usage                           |
|----------------------|---------|---------------------------------|
| React                | 18.x    | Bibliothèque principale         |
| TypeScript           | 5.x     | Typage statique                 |
| Vite                 | 5.x     | Outil de build                  |
| ESLint               | 9.x     | Linter pour le code             |
| Google Calendar API  | v3      | Synchronisation des événements  |
| Google Sheets API    | v4      | Import/Export des données       |

---

## Structure du Projet

```bash
frontend_PreparationApp/
├── public/                  # Fichiers statiques
├── src/
│   ├── assets/              # Ressources locales (CSS, images)
│   │   └── styles/          # Styles globaux et modules
│   ├── components/          # Composants React réutilisables
│   │   ├── common/          # Composants génériques
│   │   └── layout/          # Composants de mise en page
│   ├── pages/               # Pages de l'application
│   ├── hooks/               # Hooks personnalisés
│   ├── services/            # Services (API, logique métier)
│   │   └── api/             # Configuration et appels API
│   ├── types/               # Définitions de types TypeScript
│   ├── utils/               # Fonctions utilitaires
│   ├── context/             # Contexte React
│   ├── App.tsx              # Composant racine
│   └── main.tsx             # Point d'entrée
├── .gitignore
├── package.json
├── tsconfig.json
├── vite.config.ts
└── FRONTEND.md
